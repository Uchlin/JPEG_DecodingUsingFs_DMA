# Поиск пути в лабиринте на STM32H747 Discovery Kit

## Описание проекта

Программно-аппаратный комплекс для поиска оптимального пути в лабиринте на базе отладочной платы STM32H747 Discovery Kit. Система загружает изображения лабиринтов с SD-карты, выполняет их обработку, автоматически определяет точки входа и выхода, находит кратчайший путь с использованием алгоритма BFS и визуализирует результат на встроенном LCD-дисплее.

## Функциональные возможности

- **Загрузка изображений** – чтение JPEG-файлов с microSD-карты через интерфейс SDMMC
- **Декодирование JPEG** – аппаратное ускорение с использованием встроенного JPEG-кодека
- **Обработка изображения** – преобразование в бинарную карту проходимости (стены/проходы)
- **Автоматическое детектирование** – поиск точек входа и выхода на границах лабиринта
- **Поиск пути** – алгоритм BFS (поиск в ширину) для нахождения кратчайшего маршрута
- **Визуализация** – четыре режима отображения на LCD-дисплее 480×800
- **Управление** – навигация по меню с помощью встроенного джойстика
- **Индикация** – светодиоды для отображения состояния системы
- **Настройка интерфейса** – выбор цвета панелей управления из семи вариантов

## Режимы отображения

| Режим  | Описание                                                                                                                      |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **0** | Исходное изображение лабиринта                                                                            |
| **1** | Бинарная карта (черное – стены, белое – проходы)                                                |
| **2** | Бинарная карта с наложенным найденным путем                                                     |
| **3** | Исходное изображение с наложенным путем (вход – зеленый, выход – красный) |

## Требования к оборудованию

- **Плата**: STM32H747 Discovery Kit (STM32H747I-DISCO)
- **SD-карта**: microSD объемом 8–32 ГБ, отформатированная в FAT32
- **Изображения**: JPEG-файлы размером 480×480 пикселей
- **Питание**: USB Type-C (5 В)

## Требования к изображениям лабиринта

| Параметр          | Требование                                         |
| ------------------------- | ------------------------------------------------------------ |
| Формат              | JPEG (.jpg)                                                  |
| Разрешение      | 480 × 480 пикселей                                  |
| Цвет стен         | Темный (черный, темно-серый)           |
| Цвет проходов | Светлый (белый, светло-серый)         |
| Структура        | Четкие границы, отсутствие шумов |

## Сборка и загрузка

### 1. Среда разработки

- STM32CubeIDE
- STM32Cube Firmware Package H7 v1.12.0

### 2. Импорт проекта

1. Откройте STM32CubeIDE
2. Выберите **File → Import → Existing Projects into Workspace**
3. Укажите путь к папке проекта
4. Нажмите **Finish**

### 3. Сборка

- Нажмите **Project → Build All** или кнопку **Build** (молоток)
- Убедитесь, что сборка завершилась без ошибок

### 4. Загрузка на плату

1. Подключите плату к компьютеру через USB (порт ST-LINK)
2. Нажмите **Run → Debug** или кнопку **Debug** (жук)
3. Дождитесь завершения загрузки прошивки

## Подготовка SD-карты

1. Отформатируйте microSD-карту в файловую систему **FAT32**
2. Скопируйте изображения лабиринтов в формате JPEG в корневой каталог
3. Вставьте SD-карту в слот на плате STM32H747 Discovery Kit

## Управление

Управление осуществляется с помощью встроенного 5-позиционного джойстика:

| Действие            | Функция в главном меню                        | Функция в режиме просмотра              |
| --------------------------- | ---------------------------------------------------------------- | -------------------------------------------------------------- |
| **ВВЕРХ** (↑)   | Перемещение вверх по списку файлов | —                                                             |
| **ВНИЗ** (↓)     | Перемещение вниз по списку файлов   | —                                                             |
| **ВПРАВО** (→) | Загрузка выбранного файла                 | Переключение режимов отображения |
| **ВЛЕВО** (←)   | —                                                               | Возврат в главное меню                      |
| **ЦЕНТР** (●)   | Переход в меню выбора цвета               | —                                                             |
